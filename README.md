# optorch
Auxiliary second-order optimization tools compatible with PyTorch.

## Dependencies
PyTorch >= 1.0

SciPy >= 1.3.1

NumPy >= 1.17.2

(probably works with older versions too, but not guaranteed).

## Usage
See example.py
